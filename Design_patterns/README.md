## Design Patterns Exercise

### Description (EN)

This repository contains an exercise focused on design patterns.

Each participant is required to:

* Describe one design pattern assigned by the instructor.
* Implement a different design pattern based solely on the description provided by their partner.

The goal is to practice both understanding and communicating design patterns, as well as implementing them based on abstract requirements.

---

### Opis (PL)

To repozytorium zawiera zadanie dotyczące wzorców projektowych.

Każdy uczestnik ma za zadanie:

* Opisać jeden wzorzec projektowy wskazany przez prowadzącego.
* Zaimplementować inny wzorzec projektowy wyłącznie na podstawie opisu przygotowanego przez partnera z pary.

Celem zadania jest ćwiczenie zarówno rozumienia i opisywania wzorców projektowych, jak i ich implementacji na podstawie abstrakcyjnych wymagań.
